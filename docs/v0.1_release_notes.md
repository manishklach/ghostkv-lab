## What's included

- Synthetic GhostKV simulator with sketch-based bounded elimination
- GPT-2 real-attention Q/K tensor capture and validation
- False elimination frontier analysis across layers and heads
- Learned sketch projection baseline (`LearnedSketchProjection`)

## Key finding

No safe operating points found on GPT-2 at `false_elim < 5%` with
`elimination_rate > 30%`. Frontier analysis identifies Layer 0 Head 5
as the most sketch-friendly configuration. Llama-3 validation is
the next milestone.
