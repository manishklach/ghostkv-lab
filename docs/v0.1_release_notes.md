# v0.1 Release Notes

GhostKV Lab explores whether query-time bounded elimination of reconstructable KV-cache witnesses can reduce the amount of cold KV state that must be reconsidered during transformer attention, while keeping false elimination under control.

## What exists today

- a synthetic GhostKV simulator
- GPT-2 attention-tensor capture and analysis
- false-elimination frontier analysis on real attention tensors
- hierarchical elimination baselines
- reproducible CSV and plot generation

## What remains hypothetical

- production runtime benefit
- end-to-end latency improvement
- decode-kernel integration
- broad modern-model validation

## What the current findings suggest

- attention sketches can preserve coarse geometry better than exact extreme-rank structure
- false elimination is the central technical risk
- head-wise and layer-wise variability matters
- frontier analysis is more informative than single-point averages for judging feasibility

This release is intended as a research artifact, not as a deployment-ready inference system.
