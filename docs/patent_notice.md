This repository is associated with Indian provisional patent application `202641062451`.

Title:

“GHOSTKV: A SYSTEM AND METHOD FOR QUERY-TIME BOUNDED ELIMINATION OF RECONSTRUCTABLE KEY-VALUE WITNESSES IN TRANSFORMER ATTENTION MECHANISMS”

Filing date:

`2026-05-17`

## Scope Clarification

GhostKV Lab is maintained as a research and evaluation repository. The code in this repository focuses on synthetic experiments, elimination-bound studies, and illustrative KV-memory traffic modeling.

The referenced provisional filing relates to the described systems concepts around query-time bounded elimination and reconstructable KV-cache witness records. The repository itself remains a technical research artifact intended to support evaluation and discussion of those ideas.

This notice is provided for factual context. It does not add usage restrictions beyond the repository's existing [LICENSE](../LICENSE).
